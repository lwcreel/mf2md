# mf2md
Welcome to mf2md, maintained by lwcreel.

## Description
A simple CLI tool to convert .csv files exported from MacroFactor Workouts to Formatted Markdown

## Getting Started
Clone this repository using the following command:

```bash
git clone https://github.com/lwcreel/mf2md.git
```

## Usage
Via Cargo:
```bash
cargo run /path/to/input/file.csv path/to/output/dir/
```

Via Compiled Binary:
```bash
./mf2md /path/to/input/file.csv path/to/output/dir/
```
